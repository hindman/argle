Creating a new virtualenv:

    v='3.9.4'
    deactivate
    pyenv shell $v && pyenv versions
    mkvirtualenv optopus-$v
    pip install --upgrade pip
    pip install -e . -e .[dev] -e .[test]

Activate and run tests:

    - Basic:

        workon optopus-3.9.4
        pytest -s -v tests/

    - With tox:

        - Manualy:

            pyenv shell 3.9.4  3.8.9  ...
            tox

        - Invoke:

            inv tox
